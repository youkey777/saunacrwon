# relax heal ランディングページ

静的HTML/CSS/JSで構成した「relax heal」ブランドのランディングページです。ライトトーンのナチュラルウェルネス世界観とAmazon導線を想定した枠組みのみを実装しています。

## ファイル構成
- index.html : LP本体
- css/style.css : カラーパレット・レイアウト・レスポンシブ設定
- js/script.js : 軽量なインタラクション（遅延ロード/FAQなど）
- ssets/ : 画像・動画・JSONなどの静的アセット

## プレビュー方法
好きな静的サーバでルートを開いてください。例:

`ash
npx serve .
`

## メモ
- 画像/動画はダミー前提です。実アセットに差し替えてください。
- Amazonリンクや計測の流し込みは index.html 内で直接編集できます。
